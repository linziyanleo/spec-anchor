## 注意事项 / 已知约束

- **Commit message scope 白名单**：`skill | scripts | schemas | workflow | protocol | spec | trace`。其他 scope（如 `specanchor`）会被 commit-msg hook 拒绝。
- **Pre-commit blocking=true**：lint 失败 abort commit。所有新 task spec 必须含 §1.2 Hard Boundaries 段（其余 5 段 warning-only）。
- **`gh release edit --title` 副作用**：会重置 `prerelease` 为 false，必须显式带 `--prerelease=true` 兜底（已在 v0.5.0-beta.1 release 时踩过）。
- **post-commit hook**: 会异步触发本地 `github-mirror-sync local`（非 push）。pre-push 时会触发 `github-mirror-sync push` 同步到 GitHub mirror `linziyanleo/spec-anchor`。
- **远端**: origin = 阿里 GitLab；GitHub 是手动同步的 mirror，无 git remote 记录（`gh release` 走 token 认证）。
- **未 push commit**: 当前 main ahead origin 3 个 commit。下次推进前用户已选"自己决定 push 时机"。

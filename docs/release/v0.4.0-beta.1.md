# v0.4.0-beta.1 Release Note

## Summary

`v0.4.0-beta.1` is a documentation correction prerelease for SpecAnchor's agent walkthroughs.

It fixes the public Codex walkthrough so it no longer looks like a Cursor-specific install guide, and adds a Qoder walkthrough using Qoder's documented project-level Skill path.

## What Changed

- Corrected the install path and prompt wording in `examples/agent-walkthrough/codex.md`.
- Added `examples/agent-walkthrough/qoder.md`.
- Updated example indexes and usage-proof docs to include Qoder.

## Why This Release Exists

- The previous beta release shipped a public Codex walkthrough whose install path read like a Cursor-specific example.
- Public agent walkthroughs are copy-paste user-facing surfaces, so documentation drift here is worth a patch prerelease.

## Known Limitations

- This is still a beta prerelease; public interfaces may still change before `v0.4.0`.
- The change is documentation-only; it does not alter resolver or workflow script behavior.

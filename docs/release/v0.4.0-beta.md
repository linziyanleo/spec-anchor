# v0.4.0-beta Release Note

## Summary

`v0.4.0-beta` is the agent-reliability milestone for SpecAnchor.

It keeps the public alpha surface intact, then strengthens the deterministic path an AI agent follows before editing code: resolve anchors, assemble a bounded read plan, run reliability gates, and report missing coverage instead of guessing.

## What Changed

- Added `specanchor.resolve.v2` with explicit budget, missing coverage, and trace output.
- Added `scripts/specanchor-assemble.sh` to convert resolver results into an agent-readable context plan.
- Added agent contracts and walkthrough docs for Claude Code, Codex, Cursor, and Gemini.
- Added `specanchor-hygiene.sh` plus stronger `doctor` / `validate` checks for reliability and release readiness.
- Added golden replay coverage so core resolver / assembler outputs remain reproducible.

## Upgrade Notes

- Existing alpha entrypoints still work.
- `specanchor-resolve.sh --format=json` now emits `specanchor.resolve.v2`.
- New scripts: `specanchor-assemble.sh`, `specanchor-hygiene.sh`.
- `--budget` defaults to `normal`.
- This release supersedes `v0.4.0-alpha.2` as the latest public prerelease.

## Deferred / Limited Items

- No vector search, embeddings, or network RAG.
- No MCP server or web UI.
- `--diff-from` depends on local git state.

## Publish Checklist

- latest `main` CI green
- tag `v0.4.0-beta` pushed
- GitHub Release created
- README links checked
- agent reliability tests green
- consumer install assemble smoke green
